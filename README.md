### 🚀 Highlighted Projects

#### 🛡️ **DeepShield – AI Deepfake Detection System**
*PyTorch, EfficientNet-B4, FastAPI, Hugging Face Spaces, Grad-CAM, MTCNN*
- Developed a deepfake detection pipeline utilizing **EfficientNet-B4** for class prediction and **MTCNN** for robust face detection.
- Integrated **Grad-CAM** to provide visual explanations and localize manipulated/forged regions on facial images.
- Deployed the Dockerized FastAPI backend to Hugging Face Spaces, maintaining an inference latency of **under 5 seconds**.
- 🔗 **[Hugging Face Space](https://huggingface.co/spaces/paras-tripathi/deepshield)** | **[GitHub Repository](https://github.com/paras-tripathi/DeepShield)**

#### 🤖 **NOVA – AI Desktop Assistant**
*Python, Google Gemini API, NLP, OOP*
- Engineered an AI-powered desktop assistant using Python and the **Google Gemini API**, employing OOP principles for modular and clean architecture.
- Integrated NLP capability to process user queries, interact with local systems, and automate repetitive tasks.
- 🔗 **[GitHub Repository](https://github.com/paras-tripathi/NOVA)**

#### 📧 **SpamShield – Email Spam Classifier**
*Python, Scikit-learn, Streamlit, NLP*
- Built an end-to-end spam filter utilizing **Multinomial Naive Bayes** and TF-IDF vectorization on 5,172 emails, achieving a classification accuracy of **94.2%**.
- Deployed a Streamlit front-end for real-time inference.
- 🔗 **[Live Demo](https://spamshield1.streamlit.app/)** | **[GitHub Repository](https://github.com/paras-tripathi/Email-Spam-Classifier)**
